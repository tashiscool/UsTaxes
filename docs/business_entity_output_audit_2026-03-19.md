## Takeaway

The current Cloudflare path is now stronger than a simple capability matrix for
`1120`, `1120-S`, `1065`, and `1041`: it emits audited computed outputs that are
verified in service-level tests, worker-level tests, and canonical fixture-backed
parity scenarios.

That said, the business/nonprofit family is still not workbook-backed, and it
is not yet equivalent to a full rendered IRS-output parity package.
